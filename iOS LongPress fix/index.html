iOS LongPress fix Plugin Demo

The iOS LongPress fix plugin allows you to suppress the magnification glass which pops up since iOS9 when the webview is longpressed by the user.

Custom plugins don't work in the simulator and the AppBuilder companion app, so inside Telerik AppBuilder choose Run - Build - iOS - App package.

Suppressing the magnification glass

I know it's boring but there is no configuration involved in order to hide the popup when you longpress the webview.

What about 3D Touch?

Since iPhone 6S you can use 3D Touch. When applying force while pressing the screen the magnification glass may pop up. This is not automatically fixed by this plugin as it may lead to side effects (onclick breaks, ontouchend works fine)

If you want to suppress it for 3D Touch as well, add <preference name="Suppress3DTouch" value="true" /> to your config.xml.
